- Diagnosis, auth, checkout require Prisma → Postgres.

1. Admin PowerShell: `wsl --install` → reboot.
2. Start **Docker Desktop** (wait until daemon running).
3. `cd C:\AgileForum\mybmadproj && docker compose up -d`
4. Update `server/.env`:
   ```env
   DATABASE_URL=postgresql://postgres:postgres@localhost:5432/mybmadproj
   ```
